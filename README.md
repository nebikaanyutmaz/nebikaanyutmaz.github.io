## Kurulum
1. GitHub'da `nebikaanyutmaz.github.io` isimli public repository oluştur.
2. Bu ZIP'i bilgisayarında aç.
3. ZIP içindeki tüm dosyaları repository'nin ana dizinine yükle.
4. Commit changes yap.
5. Settings > Pages:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
6. Custom domain alanını şimdilik boş bırak.
